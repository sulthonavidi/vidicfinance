**VidicFinance** is a modern, privacy-focused personal finance management application designed with a mobile-first approach. Built entirely with Vanilla JavaScript, it offers a seamless and responsive experience without the need for a backend server.

🔗 **Live Demo:** [View App Here](https://sulthonavidi.github.io/vidicfinance/)

## ✨ Key Features

### 📊 Dashboard & Monitoring
- **Real-time Asset Tracker:** Live price monitoring for Bitcoin (BTC) and Gold (XAUT) powered by **CoinGecko API**.
- **Interactive Analytics:** Visual expense distribution using **Chart.js**.
- **Mobile Swipe Cards:** Native-like horizontal scroll for balance cards on mobile devices.

### 💸 Transaction Management
- **Comprehensive Logging:** Track Income, Expenses, and Transfers between accounts.
- **Payment Methods:** Distinct badges for Cash vs. Bank/Digital transactions.
- **Smart Filtering:** Advanced filtering by Month, Category, Type, and Payment Method.
- **Date Grouping:** Transactions are automatically grouped by date (e.g., "Today", "Yesterday") for better readability.

### 🛠️ Utilities
- **Budgeting System:** Set monthly limits per category with visual progress indicators.
- **Recurring Transactions:** Automate monthly repetitive transactions.
- **Data Persistence:** All data is stored locally in the browser (**LocalStorage**) with an Auto-Repair mechanism.
- **Backup & Restore:** Full JSON export/import capability to move data between devices.
- **Dark/Light Mode:** Adaptive theming for user comfort.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid/Variables).
- **Scripting:** Vanilla JavaScript (ES6+).
- **Libraries:** [Chart.js](https://www.chartjs.org/) (for data visualization).
- **API:** [CoinGecko API](https://www.coingecko.com/en/api) (for crypto/gold prices).
- **Database:** Web LocalStorage API (Client-side).
